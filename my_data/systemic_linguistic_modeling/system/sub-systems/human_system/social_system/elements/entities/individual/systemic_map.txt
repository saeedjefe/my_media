System Flow Diagram

Cognitive Systems
├── Cognitive Function System
│   ├── Thinking System
│   ├── Planning System
│   ├── Decision-Making System
│    └── Outputs: Goals, Plans, Decisions

Behavioral Systems
├── Self-Management System
│    └── Inputs: Goals, Plans, Decisions (from Cognitive Systems)
│    └── Outputs: Regulated Actions, Behavior Execution

Overall Flow:
Thinking ➔ Planning ➔ Deciding (Cognitive Systems) ➔ Self-Management (Behavioral System) ➔ Real-World Behavior
